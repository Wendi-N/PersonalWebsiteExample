1/4 Grade

In order to receive a 1/4, students should be able to complete the same requirements as Github Test 1.

See Example

2/4 Grade

In order to receive a 2/4, students should complete the same as the 1/4 grade, but also add:

  • Additional fonts from Google Fonts
  • The addition of a "Favicon"
  • The use of the <ul>, <table>, and <a> tags.
  • 5 full paragraphs of text
  • At least 2 images of the student's own creation.
See Example

3/4 Grade

In order to receive a 3/4, students should complete the same as the 2/4 grade, but also add:

  • Multiple pages using the same stylesheet.
  • At least 5 images of the student's own creation.
  • 8 full paragraphs of text.
  • A CSS animation.
See Example

4/4 Grade

In order to recieve a 4/4, students should complete the same as the 3/4 grade, but also add:

  • Twitter Bootstrap Functionality.
  • At least 5 Twitter Bootstrap componenents.
  • Working Javascript functionality (that is not already included in Twitter Bootstrap).
